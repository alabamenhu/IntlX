Rakudo Exceptions Editor

Welcome to the localization editor for Rakudo exceptions files. Please wait a few seconds for the Javascript VM version of Rakudo to load, it may take sometime.

To use this tool, simply select the appropriate .rakumod file for your language, found in the lib/Intl/X/ directory of the source (hint: you can also drag and drop it onto this window). It will be automatically loaded.
